# @cherninlab/ennodia-io

Experimental local HTTP surface for Ennodia.

Install and run it separately from the core MCP package:

```sh
bunx @cherninlab/ennodia-io
```

It exposes local provider options and a non-streaming chat-completions subset
over installed local agent CLIs. See the main Ennodia docs for current limits,
CORS posture, and examples.
